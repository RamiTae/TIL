   /**category** > get요청

   응답데이터

   ```js
   [ {categoryName}, {categoryName}, ... ]
   ```

   

   /**category**/카테고리명 > get요청

   응답 데이터

   ```js
   [
     {
       id: 질문글id,
       title: 질문글 제목,
       questionFlag: 질문글 열려있는지 닫혀있는지 상태,
       createdAt: 작성 DATE,
       username: 유저이름,
       commentsCount: 답글 수
     },
     { id: 질문글id, title: 질문글 제목, ... },
     ...
   ]
   ```

   * [x] 카테고리 테이블에 default 카테고리 입력하는 코드 추가
