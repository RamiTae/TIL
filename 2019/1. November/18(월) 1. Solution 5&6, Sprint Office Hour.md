## 1-2. 5-Q9

```js
var name = "Window";
var alice = {
  name: "Alice",
  sayHi: function() {
    alert(this.name + " says hi");
  }
};
var bob = { name: "Bob" };


alice.sayHi.bind(bob);

setTimeout(alice.sayHi(), 1000);
```

> `bind()`는 일시적이다.
