# 2. Tree

* `Tree` 클래스
  * `node` : 데이터 저장
  * `child` : array
* `.addChild(value)` : tree에 값이 `value` 인 자식 `node` 추가.
  1. 값이 `value` 인 새로운 `Tree` 생성.
  2. `this.child`에 새 `Tree`를  `push`.

- `contains(target)` : `target`이 tree에 존재하는지 확인.

타이핑하다가 손으로 하는 게 더 빠를 것 같아 손으로 썼음.
